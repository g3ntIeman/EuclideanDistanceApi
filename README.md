# EuclideanDistanceApi

Этот проект предоставляет **Minimal API** для вычисления **евклидова расстояния** между двумя точками на плоскости, используя координаты `(x1, y1)` и `(x2, y2)`.

## Ход решения

- Сначала создаем новый проект **Minimal API** с шаблоном `webapi`.
- В API создаем один **POST-метод**, который принимает координаты двух точек и вычисляет расстояние по формуле Пифагора.
- Добавляем **Swagger** для удобства тестирования API через браузер.
- Для передачи данных используем **DTO** (Data Transfer Object), чтобы упорядочить данные, которые отправляются и получаются.

## Пример выполнения запроса в формате `curl`

```bash
curl -X 'POST' \
  'http://localhost:5026/distance' \
  -H 'accept: */*' \
  -H 'Content-Type: application/json' \
  -d '{
  "x1": 0,
  "y1": 0,
  "x2": 3,
  "y2": 4
}'
```
